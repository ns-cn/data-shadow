# DataShadow

#### 项目介绍
DataShadow是一个数据比对工具，可以读取多种数据源(Excel、CSV、JSON、XML、Oracle/MySQL数据库等)的结构化数据，进行数据比对，并生成比对结果。

#### 技术架构
- 开发语言: Java 23
- GUI框架: JavaFX 21.0.2
- 项目管理: Maven
- 单元测试: JUnit 5
- 主要依赖:
  - Apache POI: Excel文件读写
  - FastJSON: JSON数据处理
  - Apache Commons CSV: CSV文件处理
  - MySQL/Oracle JDBC: 数据库连接
